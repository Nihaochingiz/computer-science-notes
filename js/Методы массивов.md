## `map` — создаёт новый массив (чистый метод)

Метод `map` проходит по каждому элементу массива и возвращает **новый массив**, элементы которого преобразованы с помощью переданной функции.

```javascript
const numbers = [5, 10, 15];

const doubled = numbers.map((n) => n * 2);

console.log(doubled); // [10, 20, 30]
console.log(numbers); // [5, 10, 15] — исходный массив не изменился
```

Результирующий массив **всегда будет такой же длины**, как и оригинальный


## `filter` — фильтрует массив (чистый метод)

Метод `filter` создаёт новый массив, оставляя только те элементы, для которых функция возвращает `true`.

```javascript
const numbers = [1, 2, 3, 4, 5];

const even = numbers.filter((n) => n % 2 === 0);

console.log(even); // [2, 4]
```


## `reduce` — сводит массив к одному значению (чистый метод)

Метод `reduce` позволяет пройтись по массиву и **накопить** какое-то итоговое значение (например, сумму или объект). Он принимает функцию, которая получает два аргумента: аккумулятор и текущий элемент.

```javascript
const prices = [100, 200, 50];

const total = prices.reduce((sum, price) => sum + price, 0);

console.log(total); // 350
```


## `reverse` — переворачивает массив (изменяет исходный)

Этот метод разворачивает порядок элементов **в самом массиве**.

```javascript
const letters = ['a', 'b', 'c'];

letters.reverse();

console.log(letters); // ['c', 'b', 'a']
```

⚠️ Будьте осторожны: массив **изменяется**.


## `slice` — извлекает часть массива (чистый метод)

Метод `slice` возвращает **новый массив**, содержащий элементы от начального индекса до конечного (не включая его).

```javascript
const items = ['apple', 'banana', 'cherry', 'date'];

console.log(items.slice(1, 3)); // ['banana', 'cherry']
console.log(items.slice(2));    // ['cherry', 'date']
console.log(items.slice(-2));   // ['cherry', 'date']
```

Исходный массив не изменяется.


## `splice` — удаляет, заменяет или вставляет элементы (изменяет массив)

С помощью `splice` можно удалять, добавлять или заменять элементы в массиве. Метод возвращает **массив удалённых элементов**.

```javascript
const colors = ['red', 'green', 'blue'];

// Вставим элемент на позицию 1
colors.splice(1, 0, 'yellow');
console.log(colors); // ['red', 'yellow', 'green', 'blue']

// Удалим 2 элемента и заменим их
const removed = colors.splice(2, 2, 'purple');
console.log(removed); // ['green', 'blue']
console.log(colors);  // ['red', 'yellow', 'purple']
```


## [Добавление/удаление элементов](https://learn.javascript.ru/array-methods#dobavlenie-udalenie-elementov)

Мы уже знаем методы, которые добавляют и удаляют элементы из начала или конца:

- `arr.push(...items)` – добавляет элементы в конец,
- `arr.pop()` – извлекает элемент из конца,
- `arr.shift()` – извлекает элемент из начала,
- `arr.unshift(...items)` – добавляет элементы в начало.
