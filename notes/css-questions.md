**1. Что такое CSS и для чего он используется?**
* Определение: Cascading Style Sheets (каскадные таблицы стилей).
* Назначение: отделение оформления от структуры (HTML), управление визуальным представлением веб‑страниц.
* Основные возможности: цвета, шрифты, отступы, позиционирование, анимации и т. д.

**2. Какие существуют способы подключения CSS к HTML‑документу?**
* **Inline‑стили** (`style` в атрибутах HTML‑элементов) — низкий приоритет, не рекомендуется для крупных проектов.
* **Внутренние стили** (`<style>` в `<head>`) — для уникальных стилей одной страницы.
* **Внешние стили** (`<link rel="stylesheet" href="...">`) — основной способ, позволяет переиспользовать стили.
* `@import` — импорт других CSS‑файлов (может замедлять загрузку).

**3. Объясните концепцию каскадности и приоритета стилей (специфичности).**
* Порядок каскада: браузерные стили → внешние/внутренние → inline‑стили → `!important`.
* Специфичность: подсчёт «веса» селекторов (ID > классы/псевдоклассы/атрибуты > теги).
* Пример: `#id .class` имеет больший приоритет, чем `.class`.

**4. Что такое блочная модель (box model) и из каких частей она состоит?**
* **Content** — содержимое элемента.
* **Padding** — внутренние отступы (между содержимым и границей).
* **Border** — граница элемента.
* **Margin** — внешние отступы (отступы от соседних элементов).
* `box-sizing: border-box` — включает padding и border в общую ширину/высоту.

**5. В чём разница между `margin` и `padding`?**
* `margin` — внешний отступ, создаёт пространство *вокруг* элемента (не влияет на фон/границы).
* `padding` — внутренний отступ, создаёт пространство *внутри* элемента (фон/границы распространяются на padding).

**6. Какие значения может принимать свойство `display`? Приведите примеры.**
* `block` — занимает всю доступную ширину, начинается с новой строки (`<div>`, `<p>`).
* `inline` — занимает только необходимую ширину, не создаёт переносы (`<span>`, `<a>`).
* `inline-block` — комбинация: inline‑поведение с возможностью задать ширину/высоту.
* `none` — полностью скрывает элемент.

**7. Как центрировать элемент по горизонтали и вертикали?**
* Горизонтально: `margin: 0 auto` (для блочных элементов с заданной шириной).
* Вертикально:
    * Flexbox: `display: flex; align-items: center; justify-content: center`.
    * Grid: `display: grid; place-items: center`.
    * Позиционирование: `position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%)`.

**8. Чем отличаются `class` и `id` в CSS?**
* `class` (`.my-class`) — может применяться к нескольким элементам, низкая специфичность.
* `id` (`#my-id`) — уникален на странице, высокая специфичность (не рекомендуется злоупотреблять).

**9. Что такое псевдоклассы и псевдоэлементы? Приведите примеры.**
* **Псевдоклассы** (состояния элементов): `:hover`, `:focus`, `:active`, `:nth-child(n)`.
* **Псевдоэлементы** (виртуальные части элемента): `::before`, `::after`, `::first-line`, `::first-letter`.

**10. Как создать адаптивный дизайн с помощью медиа‑запросов (`@media`)?**
* Синтаксис: `@media (условие) { ... }`.
* Условия: ширина экрана (`min-width`, `max-width`), ориентация, разрешение и т. д.
* Пример:
  ```css
  @media (max-width: 768px) {
    .container { flex-direction: column; }
  }
  ```
